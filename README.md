# 🚗 Sistema de Gestión de Vehículos

**Materia:** Programación Orientada a Objetos

## 📋 Descripción

Sistema que modela un taller automotriz aplicando los pilares de la POO para calcular el costo total de mantenimiento de una lista de vehículos.

---

## 🏗️ Estructura de Clases---

## ✅ Requisitos Implementados

- Encapsulamiento con doble guion bajo (`__atributo`)
- Getters y Setters con `@property`
- Herencia desde la clase `Vehiculo`
- Polimorfismo sobrescribiendo `calcular_costo_mantenimiento()`
- Función `sumar_total_mantenimiento(lista_vehiculos)` fuera de las clases
- Nombres según PEP8
